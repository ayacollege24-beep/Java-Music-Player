# Java Audio Player

A simple console-based audio player built in Java that allows users to control the playback of `.wav` audio files using simple keyboard commands.

## Features
* Controls: Play (P), Stop (S), Reset to beginning (R), and Quit (Q).
* Exception handling for missing files, unsupported audio formats, and system audio line issues.
* Uses standard Java Sound API (`javax.sound.sampled`).

## Project Structure
* `musicPlayer/` - Main package directory.
  * `Class1.java` - Source code containing player logic.
* `src/M.wav` - Target audio file path for playback.
